# Inter_2
#EDA Notebook

→This project contains a simple Exploratory Data Analysis (EDA) notebook for the Titanic dataset.
→It uses Pandas, Matplotlib, Seaborn, and Plotly to understand data through statistics and visualizations.

#Features
•Summary statistics
•Histograms & boxplots
•Correlation heatmap
•Pairplots
•Interactive Plotly charts
